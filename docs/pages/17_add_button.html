Add Button

Description

The “Add” Button moves one (1) of each selected Item in the Warehouse Inventory List to the Job Inventory List.

Usage

One or more entry must be selected in the Warehouse Inventory List. If no entries are selected in the Warehouse Inventory List, nothing will happen.

No entries need to be selected in the Job Inventory List.

A Date and Job Number must be present to add the entry to the Job Inventory List. If the Date is not present, an error window will appear. If the Job Number is not present, an error window will appear. If neither are present, both error windows will appear. After reading the error message(s), click “OK” and enter the missing job information in the Job Form.
