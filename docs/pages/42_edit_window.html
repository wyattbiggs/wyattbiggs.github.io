Edit Window

Description

The “Edit” Window is how current entries are edited in the Warehouse Inventory List.

Usage

It appears after clicking the “Edit” Button in the Inventory Window. The window will close after clicking the “Apply” Button, clicking the “Cancel” Button, or hitting the Escape key.
