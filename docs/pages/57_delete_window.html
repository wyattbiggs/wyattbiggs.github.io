Delete Window

Description

The Delete Window is how current entries are permanently deleted from the Warehouse Inventory List.

Usage

It appears after clicking the “Delete” Button in the Inventory Window. The window will close after clicking the “Yes to All” Button, clicking the “Cancel” Button, or hitting the Escape key, or when there are no more entries to be deleted.
