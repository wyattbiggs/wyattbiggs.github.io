# "Other" Tab

![Alt text](/images/image74.png "Other Tab")

## Description

The "Other" Tab displays certain attributes of the [Unit of Measure](08_unit_of_measure.md), [Bay](10_bay.md), [Shelf](11_shelf.md), [Type](13_type.md), and [Subtype](14_subtype.md) dropdown menus and allows those attributes to be edited.
