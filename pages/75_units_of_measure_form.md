# Units of Measure Form

![Alt text](/images/image75.png "Units of Measure")

## Description

The Units of Measure Form displays the full names and abbreviations for the [Unit of Measure](08_unit_of_measure.md) values for the corresponding dropdown menus and allows those to be edited.
